### 是什么
这依旧是一个可直接编译执行的代码模板。
我们可以使用`Create`函数直接调用它们，**就像普通的`MessageBoxW`一样**。

### 为什么
程序创建了两个窗口：`父窗口（让桌面降低亮度）`和`主窗口（模态窗口，用于展示消息和处理信息）`。
其中，我制作了高仿Windows 10弹窗样式的渐显渐隐效果：
用`WS_EX_LAYERED`提供的特性配合`SetLayeredWindowAttributes`函数实现设置透明度，用`CreateThread`创建线程显示效果而不干扰主消息循环。
之后，我注册了一个`FBUTTON`类，它是高仿Windows RT扁平按钮的按钮类，通过主题色计算，实现鼠标悬停、按下时的按钮显示。
它们构成了活泼可爱的小燕子。

### 怎么做
怎么做就不用说了吧？
需要注意，Create会阻塞代码运行，但是不会阻塞消息循环。它的返回值为1001-1004（分别对应四个按钮），**返回0代表未选中/超时**（需要做特别处理）。
按钮参数传入`NULL`或`L""`时不创建按钮，按钮会根据文字的宽度自动调整到最佳大小，文本和标题也是一样，文本会调整窗口高度，标题则是调整宽度。
